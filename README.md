# HearMe
Это проект на основе Raspberry pi 5. Это устройство для глухих и слабослышащих для распознавания речи, здесь вы сможете увидеть все используемые материалы, программы и зависимости. 

Для начала нужно предустановить необходимые библиотеки и нейросети. Для него я использовал следующий список:
-Vosk
-Ardos Translate

Их можно скачать используя файл requirements.txt

Далее используем файл с названием main.py
